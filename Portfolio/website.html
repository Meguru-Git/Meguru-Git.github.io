Webサイト制作

北澤焙煎珈琲

制作年：2021年 ４月

使用したフレームワーク

  • HTML5

  • CSS3

  • WordPress

コンセプト・意識したこと

隠れ家風のカフェが新たにコーヒー豆の通販をスタートしたという設定。
ターゲットは25歳の少し意識高めな会社員男性。
「自分だけが知っている美味しいコーヒー」という印象を伝えることを一番に意識して作成しました。

配色についても意識して、テーマカラーはブラウン。補色のターコイズカラーをボタンなどの強調色に使用しました。

さくらエステイト

制作年：2021年 ４月

使用したフレームワーク

  • HTML5

  • CSS3

  • WordPress

コンセプト・意識したこと

福岡の地域密着型の不動産会社をクライアントに仮定して作成。
要望は、
「カラフルで親しみやすい印象」
「会社紹介だけでなく実用性も重視」
の2つ。

上記の要望はクラウドワークスに実際に掲載のあった案件を元にしています。

簡単に部屋を選べるように細かいカテゴリ分け。
画像を大きくして部屋の写真を見やすくするといった部分にこだわりました。

20〜30代のためのおすすめワイン

制作年：2021年 5月

使用したフレームワーク

  • Bootstrap

  • CSS3

  • WordPress

コンセプト・意識したこと

20〜30代のワイン初心者をターゲットとして作成。
コンセプトは、「初心者でも直感的に選べるワインサイト」です。

BootStrapを使用しつつ、WordPressのオリジナルテーマをとして作成したサイト。
このサイトの作成を通じて、既存のサイトのWordPress化が可能になりました。
