Skill

Webスキル

HTML & CSS

1からHTML & CSSをマークアップしてサイト制作が可能です。
もちろんレスポンシブ対応も行います。
現状に留まる事なく、最新のWebトレンドを把握しながら制作を行います。

BootStrap

ハイセンスなサイトを実装するために人気のCSSフレームワークです。
世界中で人気のデザインを、お気軽に御社のWebサイトに実装可能です。

JavaScript

JavaScriptを用いて、動きのあるサイトを実装することができます。
タブUIやスライドショーなど、お客様にとって使いやすいサイトを作る上で欠かせない言語。
その可能性をさらに広げるべく、今現在も日々研究中です。

ライティングスキル

記事執筆

過去に塾で国語講師を務めていた経験から、文章作成能力には自信があります。
現在はSEOの知識を身につけ、複数のブログ運営も行っています。
持ち前の文章力を活かしたSEOライティングによって、コンテンツ制作の面でも貢献します。

WordPress

WordPressでのWeb制作が可能です。
新規での制作はもちろん、既存のサイトのWordPress化にも対応可能です。
タクソノミーやプラグインも用いてお客様が使いやすいサイト制作を実現します。
